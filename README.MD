# Application Stack:

The recommended/tested technology stack for NetWatch is...

| Service        | Technology        |
|----------------|-------------------|
| HTTP Server    | Nginx             |
| WSGI Interface | Gunicorn          |
| Application    | Python/Flask      |
| Database       | SQLite/PostgreSQL |
| Device Access  | Netmiko           |
